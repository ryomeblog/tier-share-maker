### 1.2 閲覧モード（共有URLアクセス時）

- ヘッダーに「👀 共有されたTier表を閲覧中」バナー表示
- D&D操作は無効化
- アクションボタンは「📷 画像として保存」「コピーして編集」のみ
- Tier追加/削除/名前変更/カラー変更のUIは非表示
- 編集UIトグルは非表示

## 3. 各エリア詳細

### 3.1 ヘッダー

| 要素 | 仕様 |
|---|---|
| ロゴ | 左寄せ、`text-[#e94560]`、"Tier List Maker" |
| タイトル | クリックで編集、Enter/Blur確定、Escキャンセル、max 50文字 |
| アクションボタン | flexで横並び、`bg-[#0f3460]`、モバイルでは小テキスト |

### 3.2 Tier行

| 要素 | 仕様 |
|---|---|
| Tierラベル | w-16(64px) / md:w-20(80px)、`self-stretch`で高さ自動一致 |
| カラーピッカー | 🎨ボタン（showDebugOverlay時のみ）、8色プリセット+カスタム入力 |
| アイテムエリア | flex-wrap、`min-height: 80px`、ボーダー色=Tier色40%不透明 |
| 削除ボタン | Tier行右端、`border-[#e94560]`丸ボタン |
| テキスト折り返し | `break-all`で半角文字も折り返し |

### 3.3 アイテムカード

| 状態 | 表示 |
|---|---|
| 通常 | 64×64px (`h-16 w-16`)、画像 + 下部ラベル（10px）、`border-[#555]` |
| ドラッグ中 | opacity 0.4、DragOverlayで青枠＋影付きプレビュー |
| ローカル画像（debug ON） | `border-[#FFA94D]`オレンジ枠、右上に🔒バッジ |
| ホバー | 右上に`bg-[#e94560]`の×削除ボタン表示 |
| touch対応 | `touch-action: none`、`cursor-grab` / `active:cursor-grabbing` |

### 3.4 プール（未配置エリア）

| 要素 | 仕様 |
|---|---|
| タブ切替 | 🔗 URL追加 / 📁 画像アップロード |
| URL入力 | `type="url"`、プレースホルダー、+ ラベル入力（任意） |
| ファイルアップロード | ドラッグ＆ドロップ対応、複数選択、accept="image/*"、5MB上限 |
| アイテム表示 | flex-wrap グリッド、空時は「上からアイテムを追加してください」 |

### 3.5 トースト通知

| トリガー | メッセージ | 種別 |
|---|---|---|
| URLコピー成功 | URLをクリップボードにコピーしました | success |
| ローカル画像警告 | ローカル画像N件は共有URLに含まれません... | warning |
| URL長超過 | アイテムが多すぎます。画像として保存してください | error |
| 画像保存成功 | 画像を保存しました | success |

## 4. レスポンシブ対応

| 項目 | デスクトップ (≥768px) | モバイル (<768px) |
|---|---|---|
| Tierラベル幅 | 80px (md:w-20) | 64px (w-16) |
| アイテムカードサイズ | 64×64px | 64×64px |
| ヘッダーレイアウト | 横一列 (md:flex-row) | 縦積み (flex-col) |
| アクションボタン | テキスト+アイコン (md:text-sm) | 小テキスト (text-xs) |
| ドラッグ操作 | MouseSensor (distance: 5) | TouchSensor (delay: 250ms) |
| URL入力 | 横並び (md:flex-row) | フル幅 (flex-col) |
| Tier行の間隔 | space-y-1.5 | space-y-1 |
