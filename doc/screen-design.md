### 1.2 閲覧モード（共有URLアクセス時）

- ヘッダーに「共有されたTier表を閲覧中」バナー表示
- D&D操作は無効化
- アクションボタンは「画像として保存」「コピーして編集する」のみ
- Tier追加/削除/名前変更等の編集UIは非表示

## 3. 各エリア詳細

### 3.1 ヘッダー

| 要素 | 仕様 |
|---|---|
| ロゴ | 左寄せ、アプリ名テキスト |
| タイトル入力 | 中央、クリックで編集可能、プレースホルダー「My Tier List」 |
| URLコピー | 共有URL生成→クリップボードコピー |
| 画像保存 | html-to-image でPNG生成→ダウンロード |
| リセット | 確認ダイアログ後に初期化 |

### 3.2 Tier行

| 要素 | 仕様 |
|---|---|
| Tierラベル | 幅80px固定、クリックでインライン編集、背景色付き |
| カラーピッカー | ラベル上アイコンクリックでポップオーバー表示 |
| アイテムエリア | flex-wrap、アイテムカードを横並び・折り返し |
| 削除ボタン | 右端、×アイコン。アイテムはプールに戻る |
| 最小高さ | 80px（デスクトップ）、56px（モバイル） |

### 3.3 アイテムカード

| 状態 | 表示 |
|---|---|
| 通常 | 64×64px（デスクトップ）、44×44px（モバイル）、画像+ラベル |
| ドラッグ中 | 半透明、青枠ハイライト、影付き |
| ローカル画像 | オレンジ枠、🔒バッジ表示 |
| ホバー | ×削除ボタン表示 |

### 3.4 プール（未配置エリア）

| 要素 | 仕様 |
|---|---|
| タブ切替 | 「URL追加」/「画像アップロード」 |
| URL追加 | テキスト入力+追加ボタン |
| 画像アップロード | ファイル選択/ドラッグ＆ドロップ、複数ファイル対応 |
| アイテム表示 | flex-wrap グリッド配置 |

### 3.5 トースト通知

| トリガー | メッセージ | 種別 |
|---|---|---|
| URLコピー成功 | 「URLをクリップボードにコピーしました」 | success |
| ローカル画像警告 | 「ローカル画像N件は共有URLに含まれません」 | warning |
| URL長超過 | 「アイテムが多すぎます。画像として保存してください」 | error |
| 画像保存成功 | 「画像を保存しました」 | success |

## 4. レスポンシブ対応

| 項目 | デスクトップ (≥768px) | モバイル (<768px) |
|---|---|---|
| Tierラベル幅 | 80px | 50px |
| アイテムカードサイズ | 64×64px | 44×44px |
| ヘッダーレイアウト | 横一列 | 縦積み |
| アクションボタン | テキスト+アイコン | アイコンのみ or 小テキスト |
| ドラッグ操作 | マウスドラッグ | ロングプレス→ドラッグ |
| URL入力 | 横並び | フル幅 |
